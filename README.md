# Process Blocker
A simple way to say 'stop' to unwanted processes!

# Usage
Launch: `bin/GUI.exe`

You can also run it using command line:

    call Process-Blocker.bat "processName" errorMessage delay

Example:

    call Process-Blocker.bat "chrome.exe" 1 3
